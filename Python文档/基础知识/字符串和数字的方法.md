# 字符串
* 通用的方法
```
说明：字符串是不可变的，当字符串一旦创建，就无法修改其内容。字符串属于序列的一种。

1. 获取字符串的长度。len方法
In [93]: name = 'felix zhang'

In [94]: len(name)
Out[94]: 11

In [95]:

2. 字符串拼接(+)
+：表示的是连个字符串连接起来，形成一个新的字符串。

In [97]: age = '28'

In [98]: print name
felix zhang

In [99]: name + age
Out[99]: 'felix zhang28'

In [100]:

3. 索引
从前往后：第一个字符的索引是从0开始的，最后一个索引的值是字符串的长度减1.
从后往前：第一个索引的值为索引长度(-len(string)),最后一个索引的值是-1

In [100]: name[0]
Out[100]: 'f'

In [105]: name[-1]
Out[105]: 'g'

3. 分片
示例1：整个索引的副本
In [110]: name[:]
Out[110]: 'felix zhang'

In [111]:

示例2：不包括start
In [111]: name[:2]
Out[111]: 'fe'

In [112]:
如果没有指定start，则从开始到stop，不包括stop

示例3：包括start，不包括stop
In [112]: name[3:]
Out[112]: 'ix zhang'

In [113]:
从start到最后。包括start。

示例4：反向
In [122]: name[::-1]
Out[122]: 'gnahz xilef'

In [123]:

计算反向时：新的start=-(字符串的长度减去老的start)。新的stop=-(字符串的长度减去老的stop)。步长如果为负，则表示从后往前。

```
